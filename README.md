Turn raw phone number databases (from any source) into high-quality Telegram targets.

### ⚙️ Core Features

| Feature                              | Description                                                                 |
|--------------------------------------|-----------------------------------------------------------------------------|
| Telegram registration check          | Detects whether a number is registered in Telegram                          |
| Input support                        | Any list of phone numbers (international format recommended)                |
| Account types                        | TDATA, SESSION (Telethon/Pyrogram), SESSION+JSON (auto-detected)            |
| Account-less mode                    | Check registration via proxy only (no extra profile data)                   |
| Multi-threading & high speed         | Uses multiple accounts simultaneously → massive throughput                  |
| Proxy support                        | HTTP / SOCKS5 – almost all providers; unique proxy per thread possible      |
| Flood & ban protection               | Multi-level flood control, auto-excludes restricted/banned accounts         |
| Phonebook cleaning                   | Option to clear account contacts before checking (safer & cleaner)          |
| Per-account limit                    | Set max numbers per account per cycle to avoid rate-limits                  |
| Result sorting & reports             | Valid / Invalid → separate files; TXT, CSV, XLSX export                     |
| Rich exported data                   | Phone • Telegram ID • Username • Name • Last online • Premium • Photo • Bio • Birthday • Channel • Stories |
| Performance report                   | How many numbers each account checked (per session)                         |
| Detailed per-session logs            | Full activity log saved separately – perfect for client reports             |
| GUI logging toggle                   | Turn off console logging for ultra-large datasets (faster operation)        |
| No proxy mode                        | Works without proxies (for testing / low-risk environments)                 |
| Hardware binding                     | One PC per key; free reset via manager if hardware changes                   |
| Free lifetime updates                | Regular improvements & Telegram layer compatibility                         |
| Online support                       | Fast help directly from developers                                          |
